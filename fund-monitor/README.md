# 基金净值监控 Chrome 插件

一个用于监控国内场外基金净值的 Chrome 浏览器插件。

## 功能特性

- **实时净值显示**：在浏览器工具栏显示基金最新净值
- **涨跌提醒**：当日涨跌幅超过设定阈值时发送浏览器通知
- **历史图表**：查看基金净值走势图（1周/1月/3月/1年/全部）
- **持仓收益**：计算并显示每只基金的持仓收益
- **涨跌颜色**：红色表示上涨，绿色表示下跌

## 安装方法

### 开发者模式安装

1. 下载或克隆本项目到本地
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 `fund-monitor` 文件夹
6. 插件图标将出现在浏览器工具栏

## 使用说明

### 添加基金

1. 点击浏览器工具栏中的插件图标
2. 点击右上角的「+」按钮
3. 输入 6 位基金代码（如：000001）
4. 系统会自动获取基金名称
5. 可选填持仓成本价和持有份额
6. 点击「确认添加」

### 查看详情

- 点击任意基金卡片可查看详细信息
- 包括历史净值图表、持仓收益等

### 设置提醒

1. 在基金详情中可设置涨跌提醒阈值（默认 2%）
2. 在设置页面可开启/关闭通知功能

## 数据来源

- **估算净值**：天天基金盘中实时估值（仅供参考）
- **历史净值**：东方财富基金历史数据

## 图标说明

插件提供了三种尺寸的图标：
- `icon16.png`：16x16 像素
- `icon48.png`：48x48 像素
- `icon128.png`：128x128 像素

如需自定义图标，可使用 `icons/generate-icons.html` 生成新图标，或直接替换 PNG 文件。

## 注意事项

1. 盘中时段（9:30-15:00）显示的是估算净值，收盘后显示真实净值
2. 数据每 30 分钟自动刷新一次（可在设置中调整）
3. 建议不要添加过多基金，以免影响性能
4. 网络异常时会显示缓存数据

## 技术栈

- Chrome Extension Manifest V3
- 原生 JavaScript
- Chart.js 图表库
- Chrome Storage API
